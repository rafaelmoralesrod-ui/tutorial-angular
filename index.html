Introducción

Este tutorial tiene como objetivo introducir el framework Angular, explicando sus fundamentos, proceso de instalación y primeros pasos para el desarrollo de aplicaciones web modernas.

Este tutorial ha sido realizado por Rafael Morales, estudiante de la asignatura Desarrollo de Aplicaciones en Red, como parte de una actividad académica orientada al aprendizaje de frameworks de desarrollo web.
