Primera aproximación a Angular

Una vez instalado el entorno de desarrollo, el primer contacto práctico con Angular se realiza mediante la creación de una aplicación básica, comúnmente conocida como “Hola Mundo”. Este ejemplo permite comprobar que el entorno funciona correctamente y entender el flujo básico de trabajo con el framework.

Creación de un nuevo proyecto Angular

Para crear una nueva aplicación Angular se utiliza Angular CLI desde una terminal. El siguiente comando genera automáticamente la estructura inicial del proyecto: 

ng new hola-angular

Durante la ejecución del comando, Angular CLI solicita al usuario algunas opciones de configuración, como el uso de enrutamiento o el formato de hojas de estilo.

Ejecución de la aplicación (“Hola Mundo”)

Una vez creado el proyecto, se accede a su carpeta y se lanza el servidor de desarrollo mediante el siguiente comando: 

cd hola-angular
ng serve

Angular compila la aplicación y la pone a disposición del usuario en un servidor web local accesible desde el navegador.

La aplicación puede visualizarse accediendo a la siguiente dirección: 

http://localhost:4200

Al abrir esta URL en el navegador, se muestra la página inicial generada por Angular, que constituye el primer “Hola Mundo” del framework.

Conclusión de los primeros pasos

Esta primera aproximación confirma que el entorno Angular está correctamente configurado y que el desarrollador puede crear y ejecutar una aplicación funcional. A partir de este punto, se pueden introducir conceptos más avanzados como componentes, servicios y navegación.
