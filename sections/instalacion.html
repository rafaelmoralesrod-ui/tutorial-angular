Instalación del framework Angular

¿Se precisa instalación?

Angular no requiere una instalación tradicional como una aplicación de escritorio. Se trata de un framework de desarrollo que se utiliza mediante herramientas de línea de comandos y dependencias gestionadas a través del ecosistema Node.js.

Por tanto, para trabajar con Angular es necesario preparar previamente un entorno de desarrollo adecuado, instalando una serie de herramientas que permiten crear, ejecutar y mantener proyectos Angular.

Requisitos previos del sistema

El principal requisito para trabajar con Angular es disponer de Node.js, ya que Angular utiliza npm (Node Package Manager) para la gestión de dependencias y la instalación de herramientas auxiliares.

Se recomienda instalar una versión LTS (Long Term Support) de Node.js, ya que garantiza una mayor estabilidad y compatibilidad con Angular. En el momento de la realización de este tutorial, se recomienda una versión igual o superior a Node.js 18.

Pasos para la instalación del entorno Angular

1. Instalación de Node.js

Node.js puede descargarse desde su sitio web oficial. Durante el proceso de instalación no es necesario realizar configuraciones avanzadas, ya que el instalador añade automáticamente Node.js y npm al sistema.

Una vez finalizada la instalación, se puede comprobar su correcto funcionamiento ejecutando los siguientes comandos en una terminal: 

node -v
npm -v

2. Instalación de Angular CLI

Angular CLI (Command Line Interface) es la herramienta oficial para trabajar con Angular. Permite crear proyectos, generar componentes, ejecutar servidores de desarrollo y compilar la aplicación.

La instalación de Angular CLI se realiza de forma global mediante npm, utilizando el siguiente comando: 

npm install -g @angular/cli

Tras la instalación, se puede verificar la versión instalada mediante: 

ng version

Versiones necesarias y compatibilidad

Angular mantiene una política de versiones bien definida. Cada versión principal del framework está asociada a rangos concretos de versiones de Node.js. Por este motivo, es importante asegurarse de que las versiones utilizadas son compatibles entre sí.

En este tutorial se ha trabajado con una versión reciente de Angular compatible con Node.js LTS, lo que garantiza un entorno estable y alineado con las recomendaciones oficiales.

Rutas y estructura de trabajo

No es necesario configurar manualmente rutas del sistema para trabajar con Angular, siempre que Node.js y Angular CLI se hayan instalado correctamente. Estas herramientas se añaden automáticamente a la variable de entorno PATH del sistema.

Los proyectos Angular se crean dentro de una carpeta de trabajo elegida por el desarrollador. Angular CLI genera automáticamente la estructura de directorios necesaria, por lo que no es preciso crear rutas de forma manual.

Variables de entorno

Angular no requiere la definición manual de variables de entorno para su funcionamiento básico. Sin embargo, el framework permite definir variables de entorno específicas del proyecto para gestionar distintos entornos, como desarrollo o producción.

Estas variables se gestionan mediante archivos de configuración internos del proyecto, lo que facilita la adaptación de la aplicación a distintos escenarios sin modificar el código fuente principal.

Necesidad de base de datos

Angular es un framework de frontend y, por tanto, no requiere una base de datos para su funcionamiento. En las fases iniciales del desarrollo no es necesario disponer de ningún sistema de almacenamiento de datos.

En aplicaciones reales, Angular suele comunicarse con servicios backend que gestionan el acceso a bases de datos, pero este aspecto queda fuera del alcance de la instalación y no es obligatorio para comenzar a trabajar con el framework.

Resumen del proceso de instalación

En resumen, la instalación de Angular consiste en preparar un entorno de desarrollo basado en Node.js, instalar Angular CLI y verificar que las herramientas funcionan correctamente. Este proceso es sencillo y rápido, lo que permite comenzar a desarrollar aplicaciones Angular en pocos minutos.
